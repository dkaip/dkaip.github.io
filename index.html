Java bindings to the Vulkan® graphics API

This project was created to provide Java programs access to the Vulkan® SDK available from LunarG.

Java Bindings

The following two projects:
jvulkan https::/dkaip/jvulkan The Java version of the Vulkan® API.
jvulkan-natives-Linux-x86_64 https::/dkaip/jvulkan-natives-Linux-x86_64 A native library required to use jvulkan.
provide a way for Java programmers to create applications that use the Vulkan® drivers available for their graphics card(s).
